## projects

### [ecommerce-medallion-pipeline](https://github.com/Shibin2000/ecommerce-medallion-pipeline)
Medallion architecture (Bronze → Silver → dbt Gold) for 100k e-commerce orders.
dbt mart models with schema tests, Airflow DAG, pytest suite, incremental loading.
`Python` `PySpark` `dbt` `DuckDB` `Airflow` `pytest`


### [nyc-taxi-pipeline](https://github.com/Shibin2000/nyc-taxi-pipeline)
Star schema pipeline processing 3M+ real NYC TLC taxi trips.
dim_location uses real TLC zone lookup (265 zones). MD5 hash surrogate keys. Airflow DAG.
`Python` `PySpark` `DuckDB` `Star Schema` `Airflow`

### [covid19-etl-pipeline](https://github.com/Shibin2000/covid19-etl-pipeline)
ETL pipeline on 429k rows of real COVID-19 data (Our World in Data), 9 countries.
PySpark analysis, DuckDB warehouse, Plotly dashboard, Airflow DAG.
`Python` `PySpark` `DuckDB` `Airflow` `Plotly`

### [weather-etl-pipeline](https://github.com/Shibin2000/weather-etl-pipeline)
Live weather ETL using OpenWeatherMap REST API. Loads into DuckDB, Airflow scheduled.
`Python` `REST API` `DuckDB` `Airflow`

### [spotify-etl-pipeline](https://github.com/Shibin2000/spotify-etl-pipeline)
Spotify data pipeline built in Python and Jupyter.
`Python` `Jupyter`
